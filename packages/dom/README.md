
## 下载依赖

使用npm下载

```bash
npm install @utilslib/dom
```

如何使用
```javascript
// cjs 导入方式
const utilsLibDom = require('@utilslib/dom')

// esm 导入方式
import utilsLibDom from '@utilslib/dom'
```

script标签导入方式
```html
<!-- 变量名: _utilslibDom -->
<script src="https://unpkg.com/@utilslib/dom/dist/index.global.js"></script>
```
